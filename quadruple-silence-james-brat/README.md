# James Silence Penalty — Bratty Enforcement

This is a bratty, personalised countdown timer for James' silence penalty.

## Files
- `index.html` — the countdown page. Edit the text if you want to change the wording.

## Publish on GitHub Pages
1. Create a new repository named: `quadruple-silence-chastity-lock_NumB.exe` (or any name you like)
2. Upload `index.html` and `README.md`
3. Go to Settings → Pages
4. Set Source to: `main` branch and folder `/`
5. Save

Your page will be available at:
`https://<your-username>.github.io/quadruple-silence-chastity-lock_NumB.exe/`
